View the current AI service configuration. The AI service is configured via the web dashboard (Settings → AI Service); these commands show the current state.

## config models

List enabled AI models.

```sh
volute config models
```

## config providers

List configured AI providers.

```sh
volute config providers
```

## config status

Show system configuration.

```sh
volute config status
```
